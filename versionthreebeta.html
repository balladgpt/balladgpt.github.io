BalladGPT V3 Beta

BalladGPT V3 (Beta) was released on March 19, 2023 and currently is in a testing phase. The dataset is slightly larger than V2 (1.2 MB vs 926 KB).

Running BalladGPT V3

To generate content using BalladGPT V3, just change the model name to balladgpt/balladgpt-v3-beta.

from transformers import pipeline
generator = pipeline('text-generation', model='balladgpt/balladgpt-v3-beta')
print(generator("(Your Prompt)", do_sample=True, min_length=20)[0]['generated_text'])

BalladGPT V3 is still in beta and should not be relied upon for production purposes.
