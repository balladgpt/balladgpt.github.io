Unprompted Generation

What is Unprompted Generation?

Unprompted generation is very simple: allowing the model to generate a ballad without a prompt.

Warning: Unprompted generation may generate content not suitable for all ages as this allows the AI to be more creative, generating somewhat random text.

How can I use Unprompted Generation?

To use unprompted generation, just leave the prompt blank:

1. Install Transformers

pip3 install transformers

2. Download model from Hugging Face via Transformers and generate text

from transformers import pipeline
generator = pipeline('text-generation', model='balladgpt/balladgpt-v2')
print(generator("", do_sample=True, min_length=20)[0]['generated_text'])

Running this code may generate an output such as:
